## Get started 

### CLI

Install the template:

```bash
dotnet new install Bnaya.SourceGenerator.Template
```

Create new:

```bash
dotnet new srcgen -h
dotnet new srcgen -n Comp.MyGenerator
```

Uninstall the template:

```bash
dotnet new uninstall Bnaya.SourceGenerator.Template
```
